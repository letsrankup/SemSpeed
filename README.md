# Business OS - Advanced AI SaaS Platform

An autonomous, full-stack Business Operating System built for seamless management, AI-driven automations, advanced SEO/GEO, and comprehensive CRM capabilities.

## Tech Stack
- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Database & Auth:** Supabase
- **AI Integration:** OpenRouter (Llama 3.3, etc.)

## Manual Setup Instructions
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Copy `.env.example` to `.env.local` and add your keys.
4. Run `npm run dev` to start the development server.
